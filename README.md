# Algo Hub: Core Algorithms & Visualization Engine 🧠💻

## 📌 Project Overview
This repository serves as a comprehensive collection of fundamental Computer Science algorithms implemented in Python, paired with concepts for interactive visual animations. The project demonstrates a deep understanding of algorithm design techniques, including Divide & Conquer, Dynamic Programming, Greedy strategies, and Backtracking.

Whether it's optimizing routes with the Traveling Salesperson Problem or visualizing the recursive steps of N-Queens, this hub bridges the gap between complex mathematical logic and understandable visual execution.

## ⚙️ Implemented Algorithms
The core engine includes optimized Python implementations for the following advanced algorithms:

### 1. Graph & Pathfinding Algorithms
* **Kruskal's Algorithm:** Finds the Minimum Spanning Tree (MST) using Disjoint Sets (Union-Find).
* **Floyd-Warshall:** Solves the All-Pairs Shortest Path problem using Dynamic Programming.
* **Traveling Salesperson Problem (TSP):** Finds the shortest possible route visiting multiple cities and returning to the origin.

### 2. Sorting & Divide and Conquer
* **Quick Sort:** Efficient, in-place sorting using partitioning.
* **Merge Sort:** Stable, out-of-place sorting utilizing the divide and conquer paradigm.

### 3. Dynamic Programming & Optimization
* **Rod Cutting Problem:** Determines the optimal way to cut a rod to maximize revenue using bottom-up tabulation.
* **Huffman Coding:** A greedy algorithm used for lossless data compression by assigning variable-length codes based on character frequencies.

### 4. Backtracking
* **N-Queens Problem:** Recursively places N non-attacking queens on an NxN chessboard.

## 🛠️ Tech Stack & Concepts
* **Language:** Python 3.x
* **Paradigms:** Dynamic Programming, Greedy Algorithms, Backtracking, Divide & Conquer.
* **Data Structures:** Graphs, Trees, Disjoint Sets, Matrices, Recursion Trees.
* **Visualization Concept:** Integrates with frontend animation hubs to provide real-time, step-by-step visual feedback of the algorithmic execution.
